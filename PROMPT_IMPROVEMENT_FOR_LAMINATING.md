# 라미네이팅 견적 구체적 답변을 위한 프롬프트 개선 가이드

## 🎯 문제점

현재 "라미네이팅 견적" 버튼을 클릭해도 기본적인 답변만 나오고, NotebookLM처럼 구체적인 정보가 제공되지 않습니다.

## 💡 해결 방법

프롬프트에 라미네이팅에 대한 **구체적이고 상세한 정보**를 추가해야 합니다.

---

## 📝 프롬프트에 추가할 내용

관리자 페이지(`/admin/chatbot-prompt`)에서 프롬프트를 다음과 같이 수정하세요:

### 1. 라미네이팅 견적 섹션 추가

프롬프트에 다음 내용을 **명확한 섹션**으로 추가:

```markdown
## 💰 견적 산정 기준

### 라미네이팅 견적

**기본 정보:**
- 견적은 A4 크기를 기준으로 합니다
- 총 금액이 30,000원 이하일 경우 기본 단가 30,000원이 적용됩니다

**1. 기본 단가 (A4 기준, 장당):**
- 유광 라미네이팅: 약 800원 - 1,500원/매
- 무광 라미네이팅: 약 700원 - 1,300원/매

**2. 수량별 할인:**
- 100매 이상: 10% 할인
- 500매 이상: 20% 할인
- 1,000매 이상: 30% 할인

**3. 추가 고려 사항:**
- 크기는 A4 기준으로 비례하여 계산됩니다
- 필름 종류에 따라 가격이 달라질 수 있습니다
- 견적에 필요한 정보: 인쇄물 종류 및 크기, 수량, 필름 종류(유광/무광/벨벳), 납기일

**4. 필름 종류별 특징:**
- 유광 라미네이팅: 화려하고 고급스러운 느낌, 광택 효과
- 무광 라미네이팅: 세련되고 모던한 느낌, 부드러운 표면
- 벨벳 라미네이팅: 부드럽고 고급스러운 촉감, 독특한 질감

**5. 적용 분야:**
- 책 표지, 패키지, 메뉴판, 명함, 포스터 등
- 인쇄물 표면 보호 및 질감 향상
```

### 2. 답변 가이드 추가

프롬프트에 다음 답변 예시를 추가:

```markdown
## 📋 서비스별 답변 가이드

### "라미네이팅 견적" 질문 시 답변 방법

사용자가 "라미네이팅 견적" 또는 "라미네이팅 견적이 궁금해요"라고 물어보면:

**답변 예시:**
"라미네이팅 견적에 대해 안내해 드리겠습니다.

견적은 A4 크기를 기준으로 하며, 총 금액이 30,000원 이하일 경우 기본 단가 30,000원이 적용됩니다.

**1. 기본 단가 (A4 기준, 장당):**
• 유광 라미네이팅: 약 800원 - 1,500원/매
• 무광 라미네이팅: 약 700원 - 1,300원/매

**2. 수량별 할인:**
• 100매 이상: 10% 할인
• 500매 이상: 20% 할인
• 1,000매 이상: 30% 할인

**3. 추가 고려 사항:**
• 크기는 A4 기준으로 비례하여 계산됩니다

정확한 견적을 위해서는 수량, 크기, 필름 종류(유광/무광/벨벳)를 알려주시면 더 정확한 견적을 드릴 수 있습니다."
```

## 🔧 코드 개선 사항 (이미 완료됨)

### 1. 메시지 보강 로직
- "라미네이팅 견적" 클릭 시 메시지가 자동으로 보강됨
- Gemini에게 구체적인 정보를 요청하도록 메시지 변경

### 2. maxOutputTokens 증가
- 일반 질문: 200 토큰
- 견적 질문: 500 토큰 (상세한 답변 허용)

### 3. 답변 길이 제한 완화
- 일반 질문: 200자
- 견적 질문: 500자 (상세한 정보 포함 가능)

## ✅ 프롬프트 작성 체크리스트

프롬프트에 다음 내용이 포함되어 있는지 확인:

- [ ] **라미네이팅 기본 단가** (유광/무광별)
- [ ] **수량별 할인 정보** (100매/500매/1,000매 이상)
- [ ] **필름 종류별 특징** (유광/무광/벨벳)
- [ ] **견적에 필요한 정보** (수량, 크기, 필름 종류, 납기일)
- [ ] **추가 고려 사항** (크기 비례 계산, 기본 단가 등)
- [ ] **답변 예시** (NotebookLM처럼 구체적인 형식)

## 🎯 NotebookLM과 동일한 답변을 위한 프롬프트 구조

NotebookLM이 제공하는 답변 형식을 참고하여 프롬프트를 작성하세요:

### NotebookLM 답변 형식:
```
라미네이팅 견적에 대해 안내해 드리겠습니다.

견적은 A4 크기를 기준으로 하며, 총 금액이 30,000원 이하일 경우 기본 단가 30,000원이 적용됩니다.

1. 기본 단가 (A4 기준, 장당)
• 유광 라미네이팅: 약 800원 - 1,500원/매
• 무광 라미네이팅: 약 700원 - 1,300원/매

2. 수량별 할인
• 100매 이상: 10% 할인
• 500매 이상: 20% 할인
• 1,000매 이상: 30% 할인

3. 추가 고려 사항
• 크기는 A4 기준으로 비례하여 계산됩니다
```

### 프롬프트에 포함할 내용:
1. **구체적인 숫자**: 800원-1,500원, 700원-1,300원 등
2. **명확한 구조**: 1, 2, 3으로 섹션 구분
3. **상세한 정보**: 기본 단가, 할인율, 추가 고려 사항
4. **실용적인 정보**: 크기 비례 계산, 기본 단가 적용 기준

## 📝 관리자 페이지에서 수정 방법

1. **관리자 페이지 접속**: `/admin/chatbot-prompt`
2. **수정 모드 활성화**: "✏️ 프롬프트 수정" 버튼 클릭
3. **프롬프트에 위 내용 추가**:
   - "## 💰 견적 산정 기준" 섹션 추가
   - "### 라미네이팅 견적" 하위 섹션 추가
   - 구체적인 숫자와 정보 포함
4. **저장**: "저장하기" 버튼 클릭
5. **서버 재시작** (필요시)

## ⚠️ 주의사항

### 1. 프롬프트 구조
- 섹션을 명확하게 구분 (##, ### 사용)
- 숫자와 정보를 정확하게 작성
- NotebookLM 답변 형식을 참고

### 2. 정보의 정확성
- 실제 가격 정보를 정확하게 입력
- 할인율과 기준을 명확히 표시
- 추가 고려 사항을 포함

### 3. 답변 길이
- 코드에서 maxOutputTokens: 500으로 설정됨
- 프롬프트에 충분한 정보가 있으면 긴 답변 가능
- 하지만 간결성도 유지
