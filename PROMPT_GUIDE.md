# 챗봇 프롬프트 작성 가이드

## 📋 개요

챗봇 프롬프트는 관리자 페이지(`/admin/chatbot-prompt`)에서 관리할 수 있으며, 모든 챗봇 답변의 기준이 됩니다.

## 🎯 예상질문 버튼 관련 내용 추가

현재 챗봇에는 다음 예상질문 버튼들이 있습니다:
- **상담원 연결** (`agent`)
- **파일 제출 방법** (`file`)
- **납기일 문의** (`delivery`)
- **연락처 안내** (`contact`)

이 버튼들에 대한 답변을 프롬프트에 포함시키면, 챗봇이 해당 내용을 사용합니다.

## 📝 프롬프트에 추가할 내용 예시

관리자 페이지의 프롬프트에 다음 섹션을 추가하세요:

```
📞 **연락처 안내:**
고객이 연락처를 물어볼 때는 다음 정보를 제공하세요:
- 전화: 02-1357-2468
- 이메일: info@jeongwoo.co.kr
- 주소: 서울시 XX구 XX동
- 영업시간: 평일 09:00 - 18:00
- 온라인 문의 폼: /contact
- 무료 상담 서비스 제공 중

👤 **상담원 연결 안내:**
고객이 상담원 연결을 요청할 때는 다음 정보를 제공하세요:
- 전화 상담: 02-1357-2468 (평일 09:00-18:00)
- 이메일 문의: info@jeongwoo.co.kr (24시간 접수, 24시간 이내 답변)
- 온라인 문의: /contact 페이지에서 문의 가능
- 상담원과 직접 대화하여 더 정확한 견적과 상담을 받을 수 있습니다

📄 **파일 제출 방법 안내:**
고객이 파일 제출 방법을 물어볼 때는 다음 정보를 제공하세요:
- 파일 형식: PDF, AI, EPS
- 해상도: 300DPI 이상
- 컬러 모드: CMYK
- 코팅 영역: 별도 레이어로 표시
- 이메일 제출: info@jeongwoo.co.kr (제목에 "파일 제출" 명시)
- 웹하드 업로드: https://webhard.jeongwoo.co.kr (아이디/비밀번호는 문의 시 안내)
- 온라인 문의 폼: /contact 페이지에서 파일 첨부 가능

⏱️ **작업 프로세스 및 납기일 안내:**
고객이 납기일을 물어볼 때는 다음 정보를 제공하세요:
- 작업 프로세스: 4단계 (상담 → 견적 → 작업 → 납품)
- 일반 작업: 2-3일 소요
- 긴급 작업: 당일 가능 (추가 비용 발생)
- 택배 발송: 1일 추가
- 정확한 납기일은 작업량과 난이도에 따라 달라짐
```

## 🔍 프롬프트에서 내용 찾기 방법

챗봇은 프롬프트에서 다음 키워드로 섹션을 찾습니다:
- **연락처 안내**: `연락처 안내` 섹션
- **상담원 연결**: `상담원` 키워드가 포함된 섹션
- **파일 제출 방법**: `파일 제출 방법` 섹션
- **납기일 문의**: `납기일` 또는 `작업 프로세스` 섹션

## ✅ 확인 방법

1. 관리자 페이지에서 프롬프트에 위 내용 추가
2. "저장하기" 클릭
3. 챗봇에서 예상질문 버튼 클릭하여 답변 확인
4. 프롬프트에 추가한 내용이 답변에 포함되는지 확인

## 📌 주의사항

- 프롬프트에 해당 섹션이 없으면 기본 답변이 사용됩니다
- 전화번호와 이메일 주소는 프롬프트에서 자동으로 추출됩니다
- 섹션 제목은 정확히 일치해야 합니다 (대소문자 구분 없음)
